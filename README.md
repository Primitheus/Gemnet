# Gemnet
Rumble Fighter/Gem Fighter Server Emulator.

- Features
    * Persistant Inventory
    * Shop (CASH & CARAT prices not accurate)
    * Training
    * Boss Mode
    * Survival

- In Progress
    * Lobby
    * Create/Join Rooms
    * Opening Boxes/Packages
    * Medal Shop
    * EXP/CARAT Rewards
